How to run 




1. Extract the file and copy ofrs folder

2.Paste inside root directory(wamp wamp/www)

3.Open PHPMyAdmin (http://localhost/phpmyadmin)

4. Create a database with the name  ofrsdb

5. Import ofrsdb.sql file

5. Run the script http://localhost/ofrs

6.Access admin http://localhost/admin
